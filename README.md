# Smart Text Assistant

Production-ready NLP application combining deep learning transformers (BERT, GPT-2) with statistical models for intelligent text processing and generation.

## Features

- **Smart Writing**: Auto-correction and text completion
- **Guided Writing**: Word-by-word AI suggestions
- **Ensemble Models**: BERT (110M) + DistilGPT-2 (82M) + N-gram

## Quick Start

```bash
pip install -r requirements.txt
python app.py
```

## Tech Stack

Python 3.11 | PyTorch | Transformers | BERT | DistilGPT-2 | Gradio
